## Задание 2  
Вычислить среднюю сумму чека для каждого клиента.

Пояснение логики: Запрос группирует продажи по customer_id с помощью `GROUP BY` и вычисляет среднюю сумму покупки `(AVG)` для каждого клиента. Функция `ROUND` округляет результат до двух знаков после запятой. Сортировка по убыванию показывает клиентов с самым высоким средним чеком.

Скриншот выполнения запроса (GROUP BY + AVG):
задание 3.2

## Задание 3  
Вывести `dealership_id`, продавших более 10 уникальных продуктов.

Пояснение логики: `GROUP BY dealership_id` группирует продажи по дилерским центрам, `COUNT(DISTINCT product_id)` считает количество уникальных проданных товаров для каждого дилера, `HAVING` фильтрует группы, оставляя только те дилерские центры, которые продали более 10 уникальных продуктов.

Скриншот выполнения запроса (GROUP BY + HAVING + DISTINCT):
